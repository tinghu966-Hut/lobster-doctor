🤖 模型配置 ⭐⭐⭐ 最推荐
给龙虾配上 AI 大脑，它才能跟你聊天、干活。
💡 为什么要配模型？
OpenClaw 的核心能力来自 AI 模型。不配模型，龙虾就是个空壳。 我们推荐 DeepSeek — 国内直接访问、速度快、性价比超高，最适合新手。
🔑 选择模型并填入 Key
💡 DeepSeek 好在哪里？
• ✅ 国内直接访问，不用翻墙
• ✅ 注册送 500 万 tokens，够用很久
• ✅ 速度很快，跟用国内 App 一样流畅
• ✅ 价格只有 GPT 的 1/10

去哪拿 Key？ 打开 platform.deepseek.com → 注册 → 左侧"API Keys" → 创建 Key

🔍 搜索工具 ⭐⭐⭐ 最推荐
给龙虾装个"搜索引擎"—— 它就能上网查资料了。
💡 为什么要配搜索？
没有搜索的龙虾，只能靠自己的记忆回答问题。
配了搜索，龙虾就能实时上网查最新新闻、资料、教程——就像你一边查百度一边聊天一样。

我们推荐 Tavily — 专为 AI 设计的搜索引擎，每月 1000 次免费搜索，完全够用。
🔑 配置 Tavily 搜索
三步搞定：注册 → 复制 Key → 粘贴保存
1
注册 Tavily 账号
打开 tavily.com → 点击 "Get Started" → 用邮箱注册
2
获取 API Key
登录后进入 Dashboard → 复制你的 API Key（以 "tvly-" 开头）
📖 装好搜索后的效果
✅ 龙虾可以上网查新闻、查教程
✅ 问"今天天气怎么样" — 实时告诉你
✅ 问"这个 Bug 怎么修" — 去网上找最新答案
✅ 所有搜索免费，每月 1000 次，用到吐
💬 飞书配置 ⭐⭐ 推荐
在手机上跟龙虾聊天！飞书是国内最好用的沟通工具，免费、稳定。
💡 为什么推荐飞书？
• 🇨🇳 国内直接使用，不用翻墙
• 🆓 个人和中小企业免费
• 📱 手机和电脑都能用，随时随地和龙虾聊天
• 🔗 国内网络直连，速度飞快
🔑 配置飞书机器人
在飞书开放平台创建应用，拿到 App ID 和 App Secret 填进来。
1
打开飞书开放平台
访问 open.feishu.cn → 点击"创建应用" → 选"企业自建应用"
2
获取凭证
在"凭证与基础信息"页，复制 App ID 和 App Secret 填到下面
3
发布应用
创建后记得"创建版本"并"申请发布"，审核通过才能用
