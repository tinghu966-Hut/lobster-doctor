📖 关于龙虾保姆

龙虾保姆是龙虾医生的升级版——从「急诊工具」进化为「一条龙服务」。
从安装引导到日常运维，从智能诊断到一键部署，全包。

十大模块：开机引导 · 仪表盘 · 智能安装 · 飞书助手 · 模型管家 · 技能商店 · 智能诊断 · 一键部署 · 急诊中心 · 备份医生

🚀 智能安装
自动检测环境、安装 OpenClaw、修复常见问题。
🔍 环境检测
🛠️ 安装 / 修复
💡 安装说明

1. 本工具会自动检测你的系统环境
2. 如果 Node.js 未安装，请先去官网下载 Node.js
3. 点击上方按钮自动安装 OpenClaw
4. 安装完成后配置飞书和模型即可使用
5. 首次使用建议运行开机引导向导完成全部配置

💬 飞书助手
可视化配置飞书机器人连接，无需手动编辑 JSON 文件。
📝 飞书配置
在 飞书开放平台 → 应用 → 凭证与基础信息 中获取
同一页面下方的 App Secret
📖 获取方法

1. 打开 飞书开放平台
2. 创建企业自建应用 → 填写应用名称
3. 在"凭证与基础信息"中获取 App ID 和 App Secret
4. 在"权限管理"中添加需要的权限
5. 发布应用 → 等待审核通过

💡 部署提示

Docker 部署 — 适合服务器环境，隔离运行，随系统自启动
Nginx 反向代理 — 如需绑定域名，配合 SSL 证书使用
