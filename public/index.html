📖 关于龙虾医生

龙虾医生是为中国 OpenClaw 用户打造的一站式伴侣工具。
解决新手安装难、配置复杂、崩溃不知咋办等痛点。

七大模块：智能安装 · 飞书助手 · 模型管家 · 技能商店 · 视频分析 · 急诊中心 · 备份医生

🚀 智能安装
自动检测环境、安装 OpenClaw、修复常见问题。
🔍 环境检测
🛠️ 一键安装 / 修复
💡 安装说明

1. 本工具会自动检测你的系统环境
2. 如果 Node.js 未安装，请先去官网下载 Node.js
3. 点击上方按钮自动安装 OpenClaw
4. 安装完成后配置飞书和模型即可使用

💬 飞书助手
可视化配置飞书机器人连接，无需手动编辑 JSON 文件。
📝 飞书配置
在 飞书开放平台 → 应用 → 凭证与基础信息 中获取
同一页面下方的 App Secret
📖 获取方法

1. 打开 飞书开放平台
2. 创建企业自建应用 → 填写应用名称
3. 在"凭证与基础信息"中获取 App ID 和 App Secret
4. 在"权限管理"中添加需要的权限
5. 发布应用 → 等待审核通过

🤖 模型管家
一键配置 AI 模型 API Key，支持 DeepSeek、阿里云、OpenAI。
🔑 API Key 配置
💡 推荐

DeepSeek — 国内直接访问，性价比高，适合新手
阿里云通义千问 — 国内直接访问，有免费额度
OpenAI — 需要海外信用卡，访问需要代理
