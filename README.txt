🦐 龙虾保姆 v2.0 — 安装说明
═══════════════════════════════════

📦 这是什么？
龙虾保姆是 OpenClaw 的中文伴侣工具。
不懂技术也能装 OpenClaw、配飞书、管模型。

✅ 电脑要求
• Windows 10 或 11
• 已安装 Node.js（如没有，安装程序会自动提示下载）
• 网络连接畅通

🚀 快速开始
1. 双击 run.bat → 浏览器自动打开
2. 跟着界面操作即可

📋 6大模块
🏠 仪表盘    — 系统状态一目了然
🚀 智能安装  — 一键安装 OpenClaw
💬 飞书助手  — 可视化配置飞书
🤖 模型管家  — DeepSeek/阿里云 API Key 管理
🧩 技能商店  — 一键安装扩展技能
🚑 急诊中心  — 一键抢救崩溃的龙虾
📦 备份医生  — 定时备份，防止数据丢失
🎥 视频分析  — AI 分析视频内容
🆕 引导向导  — 首次使用一步步引导

📞 遇到问题？
请联系开发者，或发消息给富贵虾。
